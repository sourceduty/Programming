Python Basics

python              - Start Python interactive shell
python script.py    - Run a Python script
python -m module    - Run a module as a script
python --version    - Show Python version

Basic Commands:
help()              - Open interactive help session
dir(object)         - List attributes and methods of an object
type(object)        - Show type of an object
exit() or Ctrl+D    - Exit Python shell

File Handling

open('file.txt')      - Open a file
file.read()           - Read entire file
file.readline()       - Read one line
file.readlines()      - Read all lines as a list
file.write(text)      - Write text to a file
file.close()          - Close a file

Data Structures

Lists:
list.append(x)    - Add x to a list
list.remove(x)    - Remove x from a list

Dictionaries:
dict.keys()       - Get dictionary keys
dict.values()     - Get dictionary values

Tuples:
tuple.count(x)    - Count occurrences of x in a tuple

Sets:
set.add(x)        - Add x to a set

Control Flow

if condition:     - Conditional execution
for item in list: - Loop through a list
while condition:  - Loop while condition is true
try:              - Try executing code
except:           - Handle an error

Python Modules

import math       - Math operations
import os         - OS interactions
import sys        - System-specific parameters
import datetime   - Date and time functions
import json       - JSON parsing
